# 🚗 Car & Bikes Information Portal

This project is a modern and responsive Vehicle Information Web Portal built using React and Tailwind CSS. It allows users to explore detailed information about cars and bikes, including specifications, pricing, mileage, engine details, and features.

## ✨ Key Features

- **Vehicle Listing Pages** - Browse comprehensive database of cars and bikes
- **Filtering & Search Functionality** - Advanced filters for easy vehicle discovery
- **Detailed Vehicle View Pages** - In-depth specifications and information
- **Side-by-Side Comparison** - Compare two vehicles to make informed decisions
- **🤖 AI Chatbot Integration** - Get instant, reliable, and personalized information about cars, bikes, and vehicle-related queries. The chatbot provides real-time assistance for user inquiries and helps with vehicle recommendations
- **Reusable Components** - Well-structured, modular React components
- **React Hooks & React Router** - Smooth navigation and state management

## 🛠 Technical Highlights

- Clean UI design with responsive layout
- Structured frontend architecture
- AI-powered assistance for enhanced user experience
- Optimized performance with React best practices

This project demonstrates strong React development skills, component architecture, and modern web development practices, making it an excellent portfolio project for React developers and full-stack engineers.

## 🚀 How to Run the Project

To run this project locally, simply use the `npm run dev` command in your terminal. Follow these steps:

1. **Install dependencies** (if you haven't already):
   ```bash
   npm install
   ```

2. **Start the development server**:
   Run the following command in your terminal:
   ```bash
   npm run dev
   ```

3. **View the app**:
   Open the Local URL provided in your terminal (usually `http://localhost:5173`) in your web browser.
