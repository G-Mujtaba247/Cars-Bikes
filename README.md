🚗 Car & Bikes Information Portal (React.js)

This project is a modern and responsive Vehicle Information Web Portal built using React and Tailwind CSS. It allows users to explore detailed information about cars and bikes, including specifications, pricing, mileage, engine details, and features.

The portal includes vehicle listing pages, filtering and search functionality, detailed vehicle view pages, and a comparison feature to compare two vehicles side by side. The application uses reusable components, React Hooks, and React Router for smooth navigation.

This project focuses on clean UI design, responsive layout, and structured frontend architecture, making it a strong portfolio project for React developers.
